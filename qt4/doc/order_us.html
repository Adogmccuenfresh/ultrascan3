UltraScan Media Distribution:

The non-commercial version of UltraScan software is distributed on the Internet for free, but can also be obtained on CDROM for a small handling fee.

The CDROM contains all of the following:

  • Installation instructions

  • UltraScan Documentation, manuals and WWW pages (html format).

  • UltraScan precompiled binaries for the following platforms:
    • Intel/Linux (tar.gz, Slackware, RedHat, Mandrake, Suse)
    • Sparc/Linux (tar.gz)
    • Linux-PPC (Macintosh Power PC)
    • FreeBSD
    • NetBSD
    • OpenBSD

  • QT library source distributions with precompiled library

  • QWT library source distributions with precompiled library

  • Stdlibc++ libraries

  • MySQL distribution with blank UltraScan database

Cost for CDROM: $50.00 (educational/nonprofit only) $10.00 shipping and handling charge (North America only, foreign orders add $10 for airmail).
