  • June 27-29, 2005: Analytical Ultracentrifugation Workshop

    Please mark your calendar for this year's Analytical Ultracentrifugation Workshop. The workshop will focus on problem solving with the analytical ultracentrifuge. The topics will range from hydrodynamic theory, experimental design, hands-on software training with UltraScan for data analysis and presentation, the new LIMS database system, and training on how to maintain the XLA/I instrument and diagnose instrument problems.

  • April 10th, 2005: UltraScan 7.1 released

    This is mostly a maintenance release with several bug fixes and also some GUI improvements.

  • February 18th, 2005: UltraScan 7.0 released

    Among the new features:

    • All new LIMS database system
    • LIMS web portal for use with Apache and PHP module.
    • LIMS C(s), C(D) and C(MW) implementation
    • New highly efficient and highly accurate Finite Element implementation for Lamm equation solution (Adaptive Space-Time Finite Element Method: ASTFEM - submitted to Biophysical Journal)
    • New License Manager simplifies dealing with UltraScan licenses
    • Documentation updates
    • lots and lots of bug fixes,
