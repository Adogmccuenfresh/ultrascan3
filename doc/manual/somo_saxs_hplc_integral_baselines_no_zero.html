SOMO HPLC-SAXS Module integral baseline tool: no Zero base option

Last updated: April 2014

If the Zero base checkbox is deselected, two vertical magenta lines and a vertical red line will be drawn on the left side of the chromatogram, and their three corresponding color-coded position fields will appear before the ones corresponding to the right-side three vertical lines. As for the right-side vertical lines, the left-side two magenta lines define the region over which the data will be averaged to set the initial baseline value, and an additional horizontal dashed orange line will be drawn indicating the trend of a linear regression done on this region. The additional left-side red vertical line defines the beginning of the data to be subjected to baseline removal.

By clicking on each field, the corresponding line can be moved across the chromatogram using the gray-shades bar-wheel which has appeared just below the graph in this panel. The green dashed line will give the trend of a linear baseline between the chosen starting and end points. The orange lines should be set to possibly match flat baseline regions at the beginning and at the end of the chromatogram. As shown in the example above, while a reasonable flat region could be found on the right-side, the left-side orange line shows that there is no flat baseline region there.

Pressing Keep once reasonable beginning and end regions have been defined will keep their parameters for the further operations involved in the integral baseline removal tool.

Cancel will remove the settings and revert to no baseline.
